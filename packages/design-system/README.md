# @efficiensee/design-system

Pegasus Design System tokens and components for Efficiensee.

## Structure

- `tokens/` - Design tokens (colors, typography, spacing)
- `components/` - React components
  - `ui/` - Base UI components
  - `charts/` - Chart components
  - `layout/` - Layout components

## Usage

```typescript
import { colors, typography, spacing } from "@efficiensee/design-system"
```

## Dark Mode Only

This design system is optimized for dark mode only.
